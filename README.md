# jobs-scheduler

**schedule job or cancel job by dynamically loading modules (.py file) from jobs directory.**


### Install python environment

* Install Python 3.9

* Install Python-pip
```
apt-get install python3-pip
```

* Install pipenv
```
pip install pipenv
```

* Install requirements
```
git clone https://github.com/mutalisk999/jobs-scheduler

cd jobs-scheduler

pipenv lock
pipenv install
```

### How To Run
```
pipenv shell
python scheduler_main.py
```
